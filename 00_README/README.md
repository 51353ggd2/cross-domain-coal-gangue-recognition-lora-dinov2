# Supplementary Material Package

Manuscript title:
A Cross-Domain Coal-Gangue Recognition Framework for Intelligent Sorting under Industrial Imaging Conditions Based on Auxiliary Source-Domain Sample Construction and LoRA-DINOv2

This package is organized as an English-only supplementary material framework for journal submission.
It is designed to support reproducibility, data transparency, and professional manuscript review.

## Package contents

- 00_README: Package-level instructions and data availability statement template.
- 01_Dataset: Dataset description, statistics, and placeholder folders for image data.
- 02_Data_Splits: Train, validation, and test split templates.
- 03_Annotations_and_Labels: Class labels and annotation format templates.
- 04_Code: Clean reference scripts and configuration templates.
- 05_Result_Data: Experimental result templates and the supplementary tables workbook.
- 06_Figures_and_Source_Data: Placeholder folders for figure source data and generated figures.
- 07_Model_Weights_Optional: Optional location for trained checkpoints.
- 08_Manuscript_Metadata: Summary of the experimental setting and reporting checklist.

## Important notes for submission

1. Replace all placeholder paths with local or repository paths before public release.
2. Add real dataset images only if you have permission to share them.
3. If the full image dataset cannot be made public, keep only representative samples and use the data availability statement provided in this package.
4. Do not include temporary training logs, cache files, IDE files, or private project directories.
5. All files in this package are intentionally named in English for academic submission.

## Recommended submission mode

Upload this archive as:
Supplementary Material

If the archive is too large, upload the dataset and code to a repository such as Zenodo, Figshare, OSF, or an institutional repository, and provide the repository DOI in the manuscript.
